Este es un sistema hecho en Java Swing y FireBird Embebido

El proyecto se llama PuntoVenta_FireBird_1, el cual pueden importar
a NetBeans para hacer cambios del código.

Cuando lo importes probablemente marque muchos errores,
lo que tienes que hacer es importar todas las librerías del
archivo Librerias.zip al proyecto.

La base de datos la puedes abrir con el SGBD FireBird utilizando
FlameRobin o coalquier otro administador de bases de datos para FireBird,
para abrir la base de datos te pedira usuario y contraseña los cuales se encuentran
dentro del archivo Base de Datos FireBird.zip

Una vez que se logre todo eso ejecuta el proyecto para que las librerías se compilen
y puedas abrir las ventanas gráficas.

Ojo, es un sistema embebido con firebir por lo tanto la conexión es de forma embebida,
si deseas que sea una conexión local con el SGBD solo cambia el tipo de conexión.
